# Safety Rules

## Allowed by Default

- Read CSV files.
- Generate derived CSV files.
- Generate markdown reports.
- Mark ads as `pause_candidate`.
- Mark creator outreach as `manual_review`.

## Requires Explicit Confirmation

- Pause or edit ads.
- Change campaign budget.
- Change bid strategy.
- Send creator DMs.
- Upload videos.
- Modify product listings.

## Report Language

Use "candidate", "recommendation", and "manual review" for account-changing operations.
