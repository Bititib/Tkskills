# Data Collection Modes

## CSV Mode

Use this when the seller has exported or manually prepared video rows. It is the default mode and requires no login automation.

## Browser-Assisted Mode

Use `competitor_collection_queue.csv` to open public account pages one by one. Fill visible metrics into `competitor_videos.csv`.

Recommended manual fields:

- video_url
- caption
- hook
- views
- likes
- comments
- shares if visible
- saves if visible
- posted_at
- hashtags
- product_angle
- format

## API Mode

Use only authorized sources:

- Display API for creators who authorized access.
- TikTok Shop/Affiliate data when available to the seller.
- Third-party data providers with lawful access.

Map all API data into `competitor_videos.csv` before analysis.
